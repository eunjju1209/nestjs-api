노마드 코더 `nestjs API` 강의 clone 코딩입니다.

간단하게 crud 만들며, 전체적인 nestjs에 대한 개념을 잡기 위해, 작업하였습니다.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
